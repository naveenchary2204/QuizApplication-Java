A console-based **Simple Quiz Application** built using **Core Java**
with OOP principles and Java Collections Framework.

---

## 📌 Project Overview

This project is **Task 4** of my Java Programming Internship.
It is a fully functional quiz system where users can select
a category, choose difficulty level, answer multiple-choice
questions and get a final score with grade and remarks.

---

## 📁 Project Structure

QuizApplication-Java/
├── Main.java          → Entry point & main menu
├── Question.java      → Question model class
├── QuestionBank.java  → All questions (HashMap + ArrayList)
└── QuizEngine.java    → Quiz logic, scoring & results

---

## ✨ Features

- ✅ 3 Categories — Java Programming | General Science | Mathematics
- ✅ 3 Difficulty Levels — Easy | Medium | Hard
- ✅ 20 Multiple Choice Questions total
- ✅ Instant feedback after each answer
- ✅ Explanation shown for wrong answers
- ✅ Auto Grading System — S / A / B / C / F
- ✅ Score, Percentage & Progress Bar at end
- ✅ Input Validation — only accepts A, B, C, D
- ✅ Clean modular code using 4 separate classes

---

## 🎮 How to Play

1. Run the program
2. Enter your name
3. Choose a category
4. Choose difficulty level
5. Answer each question with A, B, C or D
6. See your final score, grade and remarks!

---

## ▶️ How to Run

### Step 1 — Compile
javac *.java

### Step 2 — Run
java Main

| Percentage  | Grade | Remark                    |
|-------------|-------|---------------------------|
| 100%        |   S   | Outstanding! Perfect Score|

| 60% - 79%   |   B   | Good! Keep it up!         |
| 40% - 59%   |   C   | Average. You can do better|
| Below 40%   |   F   | Keep Practicing!          |
